# 💼 Rida Sersif – Developer Portfolio – React + Vite + Tailwind
## ⭐ Star this repo if you like it!

A fast, modern, and responsive developer portfolio website built using **React**, **Vite**, and **Tailwind CSS**. Designed to showcase my projects, skills, and contact information in a clean, professional format.

## 🚀 Tech Stack

- ⚛️ React (with Vite for fast dev/build)
- 💨 Tailwind CSS (utility-first styling)
- 🔧 JavaScript (ES6+)
- 💡 Optional: Framer Motion for animations
